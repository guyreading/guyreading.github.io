Abstract

Some notes on percieved weaknesses of PPO and potentially how to improve them.

List of Weaknesses

Introduction

  • The value function for a specific state will only be correct if our agent picks the same set of actions starting from that state until the end of the game. If the agent picks a different action, the development of the game will be different and rewards, thus value of the state, will be different to previously calculated. This is problematic as the agent needs to explore different actions for each state to discover the optimal action.
