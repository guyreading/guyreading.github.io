TerraBot
A Terra Mystica (board game) Faction Picker. The front end interface gives a GUI where you can see how a Terra Mystica starting state changes average final scoring for each faction. The full code can be found at the TerraBot github repository.
Remote Control Adaptor
Bose Volume Control Pod Remote Control. A stepper motor that is coupled to the Bose Volume Control Pod wheel, where you can remotely control the stepper motor to adjust volume. The stepper motor was controlled by an Arduino, which itself took volume up & down IR commands from a TV remote.
